# 🍽️ База данных "Столовая"

## Описание
База данных для автоматизации работы столовой. Содержит 21 таблицу: учёт ингредиентов, блюд, заказов, сотрудников, поставщиков, платежей и системы лояльности.

## Файлы
| Файл | Описание |
|------|----------|
| `create_tables.sql` | Создание 21 таблицы |
| `insert_data.sql` | Вставка 100 записей в каждую таблицу |
| `rename_columns.sql` | Переименование столбцов на русский язык |
| `queries.sql` | Примеры запросов |

## Установка
1. Откройте SSMS
2. Выполните файлы по порядку: `create_tables.sql` → `insert_data.sql` → `rename_columns.sql` → `queries.sql`

## Список таблиц
Должности, Сотрудники, Поставщики, Ингредиенты, Блюда, Состав_блюд, Меню, Состав_меню, Карты_лояльности, Клиенты, Столы, Заказы, Позиции_заказов, Способы_оплаты, Платежи, Смены, График_смен, Заказы_поставщикам, Позиции_поставок, Журнал_статусов_заказов, Движения_склада.
